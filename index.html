📋 Grant Code Management for CMMC Gap Assessment

▼
Admin Portal · assessment.cmmcreadynow.com/admin

Adding a Code

  • Log in to the admin portal at assessment.cmmcreadynow.com/admin
  • Click "Manage Grant Codes" in the top bar
  • Type the code name (e.g., CGA2026), add a description, and optionally set a max number of uses
  • Click "Add Grant Code"
  • The code is live immediately

Disabling a Code

  • Click the green "Active" button next to any code to toggle it off
  • Click again to reactivate it
  • Disabled codes will be rejected when someone tries to use them

Deleting a Code

  • Click "Delete" next to the code
  • Confirm the deletion
  • This is permanent

Monitoring Usage

  • The Usage column shows how many times each code has been used
  • If you set a max (e.g., 10), the code stops working after 10 uses

How Clients Use Codes

  1. Go to assessment.cmmcreadynow.com
  2. Click "Start Assessment"
  3. Enter the grant code on the first screen
  4. If valid, they proceed to the assessment

🔍 Project Spectrum Research

▼
Competitive/Partner Research · projectspectrum.io

What It Is

A free, DoD supported initiative run by the Office of Small Business Programs (OSBP) under NDAA 2019 Sec. 1644. It provides an online platform with cybersecurity information, resources, tools, and training for small and mid sized businesses in the Defense Industrial Base (DIB). The DoD promotes it as their preferred resource for SMBs pursuing CMMC compliance.

Who It Helps

  • Small and mid sized businesses (SMBs) in the DIB
  • Federal manufacturing supply chain companies
  • Universities and institutions
  • Mentor Protege Program (MPP) Pilot participants (small business protégés paired with larger DoD contractors)

What It Provides

  • Self assessments: Cyber Readiness Check that maps to CMMC/NIST controls. Gives a score and basic guidance. However, assessments are surface level and do not go deep into implementation evidence.
  • Training courses: Five video courses (~1 hour each) covering CUI, SSP, POA&M, and FOCI. Also "Bits & Bytes" short courses on individual CMMC L1 controls. Content is introductory only.
  • Info Hub: Blogs, white papers, video tips, and policies. Blog series covers all CMMC L1 controls but has known gaps (e.g., missing PAOBOAU requirements in access control).
  • Events/webinars: Occasional webinars with guest speakers from NSA, CyberAB, etc. Infrequent (roughly two per year).
  • Partner programs: Partnerships with MEP, SBA, and CyberAB.

Does It Provide Policy Documents?

No. Project Spectrum does not provide actual policy templates, SSP templates, or compliance documentation. It offers awareness and introductory training, but companies still need to write their own policies, build their own SSP, and document evidence of implementation. This is a major gap.

Does It Help with Implementation?

Minimal. The training is high level and conceptual. It introduces what controls require but does not walk companies through how to implement them in their specific environment. Independent reviewers note the content is "mediocre" and companies "may find it difficult to apply concepts to their organization." Video tips are outdated (last published 2020, still reference CMMC 1.0 naming).

💡 CGA/Capital Cyber Opportunity

Applicants who used Project Spectrum will likely have:

  • Basic awareness of CMMC requirements
  • A self assessment score (but not a thorough gap analysis)
  • No actual policy documents or SSP
  • No evidence packages for C3PAO assessment
  • No implementation guidance for their specific environment

What we can offer before and after the gap assessment:

  • Before: Our gap assessment goes far deeper than Project Spectrum's self assessment. We ask about actual processes, review logs, verify controls. This gives them a real picture vs. the surface level check they got from PS.
  • After: Policy document creation, SSP development, POA&M remediation support, implementation guidance, monthly compliance management, C3PAO readiness prep. Project Spectrum gives them awareness. We give them the execution.
  • Sub compliance: For companies evaluating their own subcontractors (like SimIS), we can assess their subs at L1/L2.
  • Tracking tools: Recommend IntelliGRC for larger orgs, CGA tracking software for smaller shops (2 to 10 employees).

Talk track: "Project Spectrum is a great starting point for awareness, but it doesn't do the work for you. It won't write your policies, build your SSP, or prepare you for a C3PAO audit. That's where we come in."
